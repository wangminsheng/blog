# 更新日志

所有重要的项目更新都会记录在此文件中。

## [1.0.0] - 2026-07-21

### ✨ 新增功能

- 初始化项目结构，使用 React + TypeScript + Vite 技术栈
- 实现 Header 导航组件，支持滚动时背景变化和移动端响应式菜单
- 实现 Hero 区域，包含动态粒子背景效果
- 实现技术栈可视化组件，包含能力雷达图和分类技术标签
- 实现项目展示组件，展示6个核心项目卡片
- 实现工作经历时间线组件，左右交替布局展示历年工作经验
- 实现视频介绍区域组件（预留位置，后续可补充视频）
- 实现联系方式组件，包含基本信息和简历下载功能
- 添加响应式设计，支持桌面端、平板端、移动端

### 🎨 设计风格

- 采用极简科技风设计
- 深色背景（#0f172a）
- 蓝色主色调（#3b82f6）
- 绿色表示成功/通过（#10b981）
- 橙色表示警告/注意（#f59e0b）

### 🛠️ 技术实现

- 使用 TailwindCSS 3 进行样式开发
- 使用 framer-motion 实现动画效果
- 使用 Chart.js + react-chartjs-2 实现雷达图
- 使用 lucide-react 提供图标

### 📁 项目结构

```
src/
├── components/          # React 组件
├── data/                # 数据文件
├── App.tsx              # 主应用组件
├── main.tsx             # 入口文件
└── index.css            # 全局样式
```
