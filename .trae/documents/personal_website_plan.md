# 个人技术展示网页 - 实现计划

## 📋 项目概述

为汪民胜创建一个**极简科技风**的个人技术展示网页，以网页为主展示平台，后续可补充屏幕录制视频。目标是让面试官通过一个链接就能快速了解你的技术能力和项目经验。

## 🎯 设计风格

- **主题**: 深色科技风（深蓝/黑色背景）
- **配色**: 主色调 #0f172a，点缀色 #3b82f6（蓝色）、#10b981（绿色）、#f59e0b（橙色）
- **风格**: 简洁、现代、专业，突出技术含量

## 📐 网页结构

### 1. Hero 区域
- 动态粒子背景效果
- 个人姓名和一句话定位
- 技术标签云展示
- 快速导航按钮（向下滚动）

### 2. 技术栈可视化
- 雷达图展示核心技术能力
- 技术分类：前端、后端、运维、工具
- 掌握程度区分：精通/熟练/了解
- 动态加载动画

### 3. 项目展示
- 卡片式布局展示核心项目
- 项目信息：名称、技术栈、描述、亮点
- 悬停效果展示更多细节
- 链接到项目代码或演示

### 4. 工作经历时间线
- 按年份展示工作经历
- 突出关键项目和成就
- 公司名称和职位
- 时间线动画效果

### 5. 视频介绍区域（预留）
- 预留视频嵌入位置
- 屏幕录制展示项目操作
- 视频封面图

### 6. 联系方式
- 邮箱、电话、微信（可选）
- 简历下载按钮
- GitHub、LinkedIn 链接

## ⚠️ 风险与注意事项

1. **数据准确性**: 确保项目描述和技术栈信息准确反映你的实际能力
2. **页面加载速度**: 使用懒加载和代码分割优化性能
3. **响应式设计**: 确保在移动端和PC端都有良好的显示效果
4. **视频资源**: 后续添加视频时注意文件大小和加载速度
5. **隐私保护**: 避免在网页上展示敏感信息

## 🎨 设计要点

- 使用深色背景营造科技感
- 蓝色为主色调，绿色表示成功/通过，橙色表示警告/注意
- 卡片设计采用圆角和阴影效果
- 动画效果简洁不夸张
- 字体清晰易读，层级分明

## 📱 响应式设计

- 桌面端（>1200px）: 三列布局
- 平板端（768px-1200px）: 两列布局
- 移动端（<768px）: 单列布局
